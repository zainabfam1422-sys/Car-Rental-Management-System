# 🚗 Car Rental Management System

A web-based car rental management system that allows users to browse, book, and manage rental cars.

## ✨ Features

- Browse Cars
- Car Booking
- Customer Details
- Rental Records
- Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
